# Librería Sampedro - Base de Datos SQL
Este proyecto consiste en el diseño e implementación de una base de datos en MySQL para la gestión de una librería.
Se trabajó sobre un modelo que incluye sucursales, autores, libros, vendedores y ventas, buscando representar una situación real de forma simple pero completa.
El proyecto incluye:
- Creación de la base de datos y sus tablas
- Inserción de datos de prueba
- Vistas para simplificar consultas
- Funciones para cálculos
- Stored procedures para automatizar consultas
- Triggers para registrar acciones automáticamente

## Uso

1. Ejecutar primero el archivo datos_libreria_sampedro.sql  
2. Luego ejecutar objetos_libreria_sampedro.sql  
De esta manera se crean las tablas, se insertan los datos y después se generan los objetos adicionales.
